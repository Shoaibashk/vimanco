# Change Log

All notable changes to the "vimanco" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [Unreleased]

### Added
- **Preset System:** Three curated presets (minimal, default, full) for quick configuration
- **Status Bar Display:** Shows active preset with click-to-update functionality
- **Reset Command:** New `vimanco.resetToDefaults` command to reset all settings
- **New Settings:** Support for surround, sneak, smartcase, ignorecase, camelCaseMotion plugins
- **Visual Mode Bindings:** Added `vimanco.visualModeKeyBindingsNonRecursive` for visual mode key customization
- **Smart Setting Override:** Individual settings now override preset values intelligently
